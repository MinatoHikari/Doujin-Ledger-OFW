# 快照

### 生成快照

生成快照的几种方法

1. 在设置界面**开启新建单据同时生成快照**，之后每次保存单据都会生成一个对应的快照。

2. 在单据详情页面上方点击相机图标。

3. 在快照列表上方点击扫描图标，扫描快照二维码。

### 同步快照

在出摊期间，完全离线的情况下，可以由点单人员生成二维码后，给顾客拍照，分拣或交付人员再从顾客那里扫码，同步快照信息。

在已有单据的情况下，可以选择用新快照覆盖旧的快照或取消本次扫描结果。

### 更新状态

在快照详情页面，点击上方的二维码按钮，可以选择同步快照和更新状态两种二维码，选择生成更新状态的二维码，则扫描二维码后不会覆盖原有单据
而是只更新单据状态。

### 区分不同摊位的二维码

目前在二维码下方会显示社团名称，后续会添加更多自定义二维码界面的方法。

### 区分同一社团内不同成员的二维码

在设置界面，每个成员设置不同的二维码前缀，设置完成后，生成的二维码的 ID 前会加上前缀。

# 货品

### 分享

在货品列表，长按某个列表项，会进入多选状态，此时勾选一条以上项目，右上角会出现二维码图标，点击即可生成货品分享二维码

### 同步

在货品列表右上方点击扫描图标，可扫描二维码或选择相册中二维码同步货品。

# 注意点

二维码也有其不足之处，那就是承载内容有限。

笔者推荐同步快照时，单条单据的货品不要超过 50 种，同时品名尽量简短。

同步货品时，如果种类非常多，最好分批次同步，比如一次同步个 15-20 条。

二维码承载内容过多而过于密集，会影响扫描的识别率，请注意。

## 备份与导出

- 导出货品

  - 在货品列表长按列表项进入选项模式，选择想要导出的商品，然后导出为 scv 格式文件或 生成二维码

- 导出全部订单

  - 在订单列表点击导出按钮，导出为 scv 格式文件

- csv 格式文件可以使用 excel 编辑 （也可以使用编辑器，vscode、sublime text、coteditor 等等, 也可使用网页在线编辑器）
  - xlsx 文件也可用类似表格格式，编辑完成后另存为 scv 文件

## 导入货品

- 在货品列表界面使用二维码扫码，扫描上述操作生成的二维码导入

- 在货品列表界面导入 scv 文件，可先导出一个 scv 文件了解大致格式，编辑完成后导入

### scv 编辑格式规范

1. 第一行为列名需要 id name price stock tags setIds remark 这些列
2. id 必须从上倒下递增
3. setIds 为套装商品包含的其他商品的 id, 多个商品用英文逗号分隔, 且套装商品的 id 必须大于包含商品的 id (也就是要放在在所有子商品的下面，例如套装包含了 id 为 1、4、5 的商品，
   那么 setIds 就填 1,4,5，套装的 id 必须填大于等于 6 的数字)

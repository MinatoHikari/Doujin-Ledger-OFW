# 快照

### 通过单据手动生成快照

上一篇中我们在新建单据的时候看到在生成单据的同时，成功提示中会有一个二维码按钮。这个按钮是用来生成快照的，快照可以帮助我们记录单据中的物品是否已经配好送出。

假设摊位现在只有摊主一人，快照的数据是不需要二维码传递的，豆津账本开发时已经考虑到这点，可以通过保存的单据直接生成快照。

进入对账界面，查看其中一个单据的详情，右上角有一个相机按钮，这个就是生成当前单据的快照的按钮，我们现在就生成一个快照试试。

快照生成以后，进入快照页面，可以看到刚才生成的单据的快照已经添加到列表中，状态是新快照。

### 标记货品

点击新快照右侧的按钮，可以进入快照详情页面，详情页的页面下方的货品列表会显示每件东西需要几个，已经完成配货的货品，可以点击一下列表项标记完成，标记完成的列表项背景色会改变。

同时，上方的标题卡片右下角的计数器的分子会加 1，对勾的颜色也会变化，没有全部标记对勾为黄色，全部标记对勾为绿色，全部没标记为灰色。

上方的垃圾桶按钮可以删除快照，二维码按钮则可以把快照再次分享给其他成员。

### 变更快照状态

单人情况下快照的应用就是这些，但其实，豆津账本的快照系统是为多人协作设计的，下面的章节着重介绍的就是多人协作下如何用豆津账本提升订单处理效率，从而加快排队速度，让顾客在更短的时间内，逛到更多的摊位。

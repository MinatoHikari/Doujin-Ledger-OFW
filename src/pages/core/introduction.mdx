# 前言

豆津账本是一个专为即卖会摊主而生的软件，可用于在漫展出摊的场合进行价格计算、保存交易历史、团队协同、销售额统计、商品分类统计。

笔者因为一次参与 CP 出摊的经历萌生的编写本软件的想法，相信很多摊主应该也有同样的需求痛点，但是苦于没有专门为此类场景设计的记账软件使用，
所以豆津账本应运而生。

本软件最初其实是网页+后端的一套程序，但是考虑到部署的门槛和漫展的网络状况，在有了 cp 的实际出摊经历以后决定重构成移动端 App

### 软件特色：

- 基于出摊经历，专为摊主设计。
- 没有复杂的配置，简单易上手。
- 小巧的安装包体积，齐全的功能。
- 同时支持使用远程数据库和本地数据库，可以完全离线使用。
- 快照系统，支持点单、分拣分离，达成离线状态下的多人协作。
- 支持离线状态同步货品信息。

## 开始使用

### 首次打开软件

首次打开软件，会显示基础设置画面：

![alt welcome](/welcome.png)

如图所示，需要设置社团名称和数据库类型、名称。

数据库名称可以随意取（请使用英文字母命名），但是需要注意的是之后虽然可以修改，但是新建不同名的数据库是不会迁移之前数据库的数据的，将会完全新建，请注意。

社团名之后可以自行修改，但是建议一开始就好好命名。之后社团名将会显示在各种二维码的下方，用以区别不同的摊位的二维码，非常重要，所以
建议一次就设置好合适的名字。（例如摊位号+社团名称）

数据库不出意料大多数人会选择本地数据库，之后的教程也会以此为准，但是远程数据库下软件的功能也是同样完备的，有能力或有需求的团队可以参阅——[部署并使用远程数据库](/advanced/remote-database)。
